Dataset Input:
├─ dataset_path (text)
│  └─ Example: D:/photos/bottles OR //server/share/images OR /Users/me/dataset
├─ lora_token (text, default: "sks_fragrancebottle")
├─ model_base (dropdown)
│  └─ Options: SDXL, SD1.5, Flux, flux_kontext, wan22, hunyuan, qwen_image
└─ training_type (dropdown)
   └─ Options: character, object, style, scene, motion_lora

Optional Advanced:
├─ custom_config (long text, JSON)
│  └─ If provided, merge with template
└─ validation_prompts (long text, one per line)
   └─ If empty, auto-generate from captions
